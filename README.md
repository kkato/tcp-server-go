# tcp-server-go

GoでTCPサーバー・クライアントを実装したプロジェクトです。

[Udemy - Goで作る大規模アプリケーション](https://www.udemy.com/course/go-daian)の一環として作成しました。

## 概要

このプロジェクトは以下の2つのコンポーネントで構成されています：

- **サーバー** (`main.go`): localhost:8080でリッスンし、クライアントからのリクエストを受け付ける
- **クライアント** (`client/main.go`): サーバーに接続し、複数のリクエストを送信する

## 機能

### サーバー
- TCP接続を受け付ける (localhost:8080)
- クライアントからのリクエストを受信
- JSONフォーマットでレスポンスを返す
- `"close"`リクエストを受け取ると接続を閉じる
- 複数のクライアントを並行処理 (goroutine)

### クライアント
- サーバーに接続
- 複数のリクエストを順次送信 (request0, request1, request2, close)
- サーバーからのレスポンスを受信・表示

## 使い方

### サーバーの起動

```bash
go run main.go
```

### クライアントの起動

別のターミナルで：

```bash
go run client/main.go
```

## 使用ライブラリ

- `net`: TCP接続の確立と通信
- その他のGo標準ライブラリ
